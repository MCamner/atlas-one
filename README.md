# Atlas Studio v12 Mac

Webserver edition with Prompt-OS aware routing.

## Features
- Local Java webserver on `127.0.0.1:8765`
- Prompt library loader from `web/prompts.json`
- Auto route preview for `/atlas`
- Workflow detection
- Pipeline visualizer
- Background picker with persistence
- Local project save/load
- ChatGPT handoff buttons

## Requirements
- Java 17+

## Run
```bash
./build_and_run.sh
```
Or:
```bash
./run_mac.command
```

## Package on Mac
```bash
./package_mac_app.sh
```
